### Installation
1. `git clone https://github.com/D35YNC/tg-tray-icons.git`
2. `cd tg-tray-icons/%KDE-THEME-NAME%`
3. `sudo mv telegram-* /usr/share/icons/%KDE-THEME-NAME%/64/`
4. succ
